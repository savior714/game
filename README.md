# 🎮 어린이 학습 게임 놀이터 (Learning Playground)

아이들의 성취감을 최우선으로 하는 지능형 적응형 학습 플랫폼입니다.

## 🚀 주요 기능

- **4대 과목 지원**: 국어 놀이, 수학 놀이, 영어 놀이, 과학 놀이(신설).
- **7단계 정밀 난이도**: '입문'부터 '전설'까지 아이의 실력에 맞춰 세밀하게 조정됩니다.
- **단계별 마스터 엔진 (Per-Level Mastery)**: 현재 단계를 90% 이상 완벽히 익혀야 다음 단계로 한 칸 이동합니다.
- **실시간 컨디션 케어 (Confidence Recovery)**: 2번 연속 틀리면 난이도를 1단계 하향하여 자신감을 회복시켜 줍니다.
- **중복 방지 엔진 (Anti-Repetition)**: 최근 10문제를 기억하여 동일하거나 유사한 문제가 연달아 나오지 않게 차단합니다.
- **로켓 스테이크 보상**: 20연속 정답 시 로켓 발사 애니메이션과 함께 보상 룰렛(유튜브, 간식 등)이 제공됩니다.

## 🛠️ 실행 방법

1. Python이 설치된 환경에서 다음 명령어로 서버를 실행합니다:
   ```bash
   python serve_game.py
   ```
2. 웹 브라우저에서 `http://localhost:3000` 접속 시 메인 페이지가 나타납니다.

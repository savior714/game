# Spec: Marble Drop Position & Animation Improvement

## 1. 개요
현재 마블 게임은 구슬이 항상 중앙에서 떨어지는 것처럼 보이며(미리보기 고정), 터치 지점에 따른 피드백이 부족함. 이를 개선하여 조작감을 향상시키고 시각적 즐거움을 제공함.

## 2. 주요 변경 사항
- **다이내믹 미리보기**: 다음 떨어질 구슬이 마우스/터치 위치를 실시간으로 추적함.
- **부드러운 이동 (Lerp)**: 구슬이 즉각적으로 이동하는 대신, 부드럽게 따라오도록 보간 처리함.
- **가이드 라인**: 구슬이 떨어질 궤적을 점선으로 표시하여 정확한 조작을 도움.
- **드롭 애니메이션 피드백**: 구슬을 놓는 순간 시각적 효과 추가.

## 3. 기술적 상세
- `_mbPreviewX`: 현재 미리보기 위치를 저장하는 전역 변수 추가.
- `_mbTargetX`: 목표 지점을 저장하고 `_mbPreviewX`가 이를 부드럽게 추적함.
- `onMbMove`: `mousemove`, `touchmove` 이벤트를 처리하여 `_mbTargetX` 업데이트.
- `_mbRender`:
    - `_mbPreviewX` 위치에 반투명 구슬 렌더링.
    - 구슬 하단으로 `setLineDash`를 이용한 수직 가이드 라인 렌더링.
- `onMbInput`: 클릭 시 `_mbPreviewX` 위치에서 구슬 생성.

## 4. 제약 사항
- 파일 크기 500라인 준수 (현재 약 200라인으로 여유 있음).
- 성능 최적화: `requestAnimationFrame` 루프 내에서 효율적인 렌더링 유지.

# 영어 게임 스펙

**소속**: SDD — 어린이 학습 게임 플랫폼
**상태**: ✅ 완성 | **대상**: 초등 저학년
**최종 수정**: 2026-03-29

---

## 1. 개요

영어 단어를 게임 형태로 훈련하는 애플리케이션.
이모지를 활용한 시각적 단서와 4지선다 버튼, 120초 타이머, 세션 10문제로 구성.
카테고리별 정답률과 응답 속도를 누적 분석하여 난이도를 자동 조정한다.
공통 플랫폼 아키텍처(로켓 스트릭, 적응형 난이도, 강화학습, 중복 방지)를 상속한다.
- **중복 방지**: 최근 10단어를 기억하여 연속 출제 차단 (`RECENT_LIMIT=10`)
- **주간 시험 단어**: 보호자가 등록한 단어(Weekly Words)를 60% 확률로 상시 주입 (2026-03-30 추가)

## 2. 문제 유형

| ID | 유형 | 질문 화면 | 보기 형식 |
|----|------|-----------|-----------|
| `kor2word` | 한국어 → 영어 | 이모지 + 한글 뜻 | 영단어 4지선다 |
| `spelling` | 철자 완성 | 이모지 + 한글 힌트 + `c_t` 또는 `c_ _t` | 빠진 글자 4지선다 (1자 또는 2자) |

**출제 유형 비율 (레벨별)**:
- Level 1: kor2word 60%, spelling 40%
- Level 2: kor2word 45%, spelling 55%

## 3. 단어 데이터베이스 스키마

```
WORDS = {
  {category}: {
    label: string,   // 카테고리 한글명
    icon: string,    // 카테고리 대표 이모지
    words: [en, ko, emoji, level][]  // 단어 배열 (compact)
  }
}
```

**카테고리 (10개)**: `animals`, `fruits`, `colors`, `numbers`, `body`, `space`, `jobs`, `transport`, `places`, `nature`

**난이도 기준 (7단계)**:
- **Level 0~1**: 기초 단어 (tiger, apple, 3-5자)
- **Level 2~3**: 일상 단어 (giraffe, strawberry, 6-8자)
- **Level 4~6**: 실생활 빈도가 높지만 철자가 길거나 복잡한 단어 (dinosaur, environment, 9자+)

## 7. 보기 생성

**단어 유형 (`makeWordChoices` — kor2word)**:
- 전체 단어 풀에서 정답 영단어 제외 후 중복 제거
- 랜덤 셔플 후 3개 선택 → 정답 포함 4지선다

**철자 유형 (`makeSpellingChoices` — spelling)**:
- 단어 길이 ≥ 5자이고 50% 확률: 빈칸 2개 (순차 입력)
- 그 외: 빈칸 1개 (1자 정답)
- 1자 정답: 모음이면 다른 모음, 자음이면 다른 자음 우선 선택 → 4지선다
- 2자 순차 입력 (설계 결정 2026-03-22): 빈칸을 동시 보기 대신 순서대로 노출.
  첫번째 공란(파란 밑줄+깜박임) → 1자 4지선다 → 정답 선택 시 초록으로 채워지고
  두번째 공란 활성화 → 새 1자 4지선다. 오답 또는 타임아웃 → 즉시 문제 종료.

```js
## 8.5 주간 시험 단어 우선 출제 (Weekly Prioritization)

보호자가 입력한 단어(`englishWeeklyWords`)를 집중적으로 노출하여 학습 효과를 극대화한다.

- **발동 조건**: `weeklyWords.length > 0` 이고 `Math.random() < 0.6`.
- **선택 로직**:
  1. `wrongPatterns` 중 주간 단어가 있으면 해당 단어를 최우선 선택.
  2. 없으면 주간 단어 풀에서 무작위 선택.
- **중복 방지**: 최근 단어 체크(`RECENT_LIMIT`)를 단어 수에 맞춰 `floor(N/2)`로 자동 조절하여 노출을 극대화함. (단어 수가 1개면 매회 출제 가능)
- **유형 교차**: 동일 주간 단어 재출제 시 `kor2word`와 `spelling` 유형을 강제로 교차하여 지루함 방지 및 다각도 학습 유도.
- **레이아웃**: 긴 단어(11자 이상) 출제 시 `.q-long` 클래스를 통해 폰트 크기를 자동으로 축소(`2rem` -> `1.6rem`)하고 `flex-wrap: nowrap` 및 간격 축소(`gap: 4px`)를 적용하여 한 줄 노출을 보장함 (2026-03-31 개선).
```
